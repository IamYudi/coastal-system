# Coastal Tactical Intelligence System
**Author:** Yudhveer mahajan | **Status:** Under Making

## Mission
A geographical safety and logistics tool mapping coastal zones in India. 
Starting with Tamil Nadu (Chennai) and expanding to Goa.

## Technical Stack
- **Languages:** HTML5, CSS3, JavaScript
- **Frameworks:** Bootstrap 5
- **Mapping:** Leaflet.js (OpenStreetMap)

## Features
- Rip current identification for River Mouths.
- Tactical advice for surfers and trekkers.
- Responsive design for mobile field-use.

# Contributing to Coastal-System

First off, thank you for considering contributing to the Coastal Tactical Intelligence project!

### How to Contribute
1. **Reporting Bugs:** Open an issue with a clear description and steps to reproduce.
2. **Feature Requests:** Use the Issue Templates to suggest geospatial or logic improvements.
3. **Pull Requests:** - Fork the repo.
    - Create a feature branch.
    - Ensure your code follows the tactical brevity style for AI prompts.
    - Push to your fork and submit a PR.

### Environment Setup
- Python 3.9+
- `google-generativeai`
- `gradio`
